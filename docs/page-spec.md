# 首版页面规格

本文件来自已确认的 `xingtu-short-video-bi` 页面规格。

核心方向：

- 浅色飞书企业 BI 版。
- 首屏管理驾驶舱。
- 下方曝光榜单、账号表现、演员表现。
- `/` 和 `/admin` 保持星途短视频 BI 向后兼容。
- `/hub` 为 BI Hub 入口。
- `/xingtu` 和 `/admin/xingtu` 为星途短视频模块入口。
- `/oae` 和 `/admin/oae` 为 OAE 模块预留入口。
- 不做深色大屏。
- 不做复杂权限模型。
- 适配飞书内嵌 iframe，不依赖固定最小页面宽度。
- 页面不展示 sheet、字段来源、缺失字段等技术性解释。
- 超过五位数的数据按 `万` 展示，例如 `12,226,000` 显示为 `1222万`。
- 曝光榜单包含单条视频的 Top1 和 Bot1，按视频曝光量计算，卡片优先展示视频标题。
- OAE 字段口径未确认前，不套用星途短视频指标、图表或上传解析器。

## 数据验收目标

- 总发布条数：`320`
- 总曝光量：`12,226,000`
- 整体 5S 完播率：`14.1%`
- 有演员视频条数：`71`
- 参与演员人数：`10`
- 视频号 5S：`未提供`
- 演员贡献曝光量：不平摊
